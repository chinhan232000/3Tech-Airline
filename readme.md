# Project require

- node [Version LTS](https://nodejs.dev/)

# Clone project

```
$ git clone https://github.com/chinhan232000/3Tech-Airline.git
```

# Install packages

```
$ npm install
```

# Run project

```bash
# Run dev
$ npm run dev

# Start project
$ npm run build
$ npm start
```

- View project on [`http://localhost:3000`](http://localhost:3000)
